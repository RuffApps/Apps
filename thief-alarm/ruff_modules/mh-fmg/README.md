# MH-FMG

## Description
This is the driver for `MH-FMG` Buzzer module with GPIO interface.
## Usage

* buzz

```javascript
    $('fmg').buzz();
```

* unbuzz

```javascript
    $('fmg').unbuzz();
```

## API

### Commands
* **buzz**
Buzzer sounds buzz.

	* Returns
		* (undefined)

* **unbuzz**
Buzzer silence.

	* Returns
		* (undefined)

* **turnOn**
turnOn the Buzzer.

	* Returns
		* (undefined)

* **turnOff**
turnOff the Buzzer.

	* Returns
		* (undefined)
